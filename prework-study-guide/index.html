CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

GIT

  • git status: checks out what branch you are currently on (sanity check)
  • git checkout -b branch-name: creates a new branch and switches to it
  • When pushing work:
    • *First, get your work onto your branch
    • $git add .
    • $git commit -m "thoughtful message"
    • *If you are not pushing your work, stop here
    • *Second, make sure your local main branch is up to date and push your working branch
    • $git pull origin main
    • $git push origin {branch name}
    • *Finally, go to github and merge the pull request, then clean up your local branch
    • $git checkout main
    • $git pull origin main
